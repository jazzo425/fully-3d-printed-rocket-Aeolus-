# fully 3d printed PLA rocket (Aeolus)
this project is a fully 3d printed minimum diameter rocket airframe and motor securing and nosecone to fit with a standard Quest D20W-8 and made on a bambu a1 mini

## parts list
- **3d printer access**
- **3.2mm  nylon coated, kelvar core shockcord**
- **nomex patch 300 x 300** (for protecting the streamer at deployment **size is a massive overestimate**)
- **one 3mm x 10mm woodscrew** (bottom launch lug)
- **one 3mm x 30mm woodscrew** (top launch lug)
- **Quest d20w - 8** (8 second delay) [where I bought my motor (uk)](https://wizardrockets.co.uk/product/quest-d20-6w/)
- **ripstop nylon fabric** (60mm by 2500mm streamer size)
- **access to a sewing machine**
- **epoxy** (my recommendation is jb weld or gorilla 5 mins two part epoxy) (optional)

 ## files include in project
 - openrocket design 
 - openrocket after build (adjusted weights from my build)
 - .STL for all 3d priced components:

 ## .STL file list 
 - fully assembly
 - nose cone
 - body tube and fins
 - motor retainer
 - launch lugs

## printer settings (_bambu lab focused_):
- 90% infill concentric infill pattern
- 0.08 layer line height (extra fine setting bambu)
- no surpport (**apart from nose cone and for that supports on build plate only**)
- wall loops 5
- Auto circle contour-hole (bambu setting in quality)
- nozzle temp 240 (temp settings for PLA, and PLA silk)
- bed temp 60 (smooth PEI plate)
- no extra bed adhesion needed
